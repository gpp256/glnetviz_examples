Setup
-----

To install, run the following commands.

    e.g.  
    $ sudo find . -type d -depth 1 \! -name '.*' -exec cp -r {} ${glNetViz_INSTALL_PATH}/examples/ \;
    $ sudo find  ${glNetViz_INSTALL_PATH}/examples -name "*.cgi" -exec chmod 755 {} \;
